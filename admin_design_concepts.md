# HDmobil Admin Panel - 3 Dizajnové Koncepty

## Slide 1: Titulná strana
**Nadpis:** HDmobil Admin Panel
**Podnadpis:** 3 Vizuálne návrhy pre modernú administráciu

## Slide 2: Koncept 1 - "Minimalist Green"
**Štýl:** Čistý minimalistický dizajn
**Hlavná farba:** Emerald green (#22c55e)
**Charakteristiky:**
- Biely pozadím s jemnými zelenými akcentmi
- Outline ikony (Lucide)
- Veľké karty so štatistikami
- Area chart pre tržby
- Donut chart pre kategórie
- Sidebar s hover efektami
**Vhodné pre:** Profesionálne použitie, prehľadnosť

---

## Slide 3: Koncept 2 - "Dark Mode Pro"
**Štýl:** Tmavý profesionálny dizajn
**Hlavná farba:** Dark slate (#1e293b) s emerald akcentmi
**Charakteristiky:**
- Tmavé pozadie s kontrastnými kartami
- Neon zelené akcenty
- Gradient buttons
- Glassmorphism efekty na kartách
- Animované grafy
- Kompaktný sidebar
**Vhodné pre:** Nočné použitie, moderný vzhľad

---

## Slide 4: Koncept 3 - "Gradient Fresh"
**Štýl:** Svieži gradient dizajn
**Hlavná farba:** Zeleno-tyrkysový gradient
**Charakteristiky:**
- Gradient header a sidebar
- Biele karty s tieňmi
- Rounded corners všade
- Farebné ikony v kartách
- 3D efekty na tlačidlách
- Floating action button
**Vhodné pre:** Mladistvý, energický vzhľad
